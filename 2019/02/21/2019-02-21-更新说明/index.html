更新说明

没想到吧！我胡汉三又回来啦！

一只萌新诞生在 2017 年 2 月，也就是说这个域名和之前使用的主机续费时间也是在二月。今年趁这个机会，对博客进行了一次比较大的改版。首先是为了省下一半的钱使博客更加轻量化，不再续费主机及使用 Wordpress，而是改用 Hexo，并将其托管在 Github Pages 上。使用 Github Pages 的考量在于其网络比之前的香港主机稳定很多，站长我就不用考虑如何加速国内访问了，正常情况下国内各地到 Github 的网络都是没问题的；但是，上述只是『正常情况』，Github 向来不受赵家待见，历史上也出现过短暂的封锁事件。不过作为世界上最大的代码存放网站，Github 对国内的互联网行业极为重要，在可预见的将来不太可能被全站永久封锁（当然宪都能修，这也不是不可能发生，真到了那一天再寻找别的去处吧）。不过，不被封锁不代表访问非常顺畅，偶尔国内访问 Github 也会出现缓慢、不稳定等情况，若不幸遇上请切换网络环境或稍后再试。

至于 Hexo，站长对其早有耳闻，这是一个快速、简洁且高效的博客框架。相比于 Wordpress 的全面而缓慢，Hexo 操作起来略微更加困难，然而更加高效，即使数百篇文章也能在几秒钟之内生成完毕（听说文章数上万之后 Hexo 也会变得很缓慢，不如 Hugo 快速，但估计一只萌新的文章永远达不到这一数量）。此外，不同于 Wordpress 需要登录后台管理， Hexo 更多的是在本地编辑，完成后简单几步操作发布文章即可。这样站长能专注于本地 Markdown 文件，不再有面对本地与 Wordpress 数据库两部分文章的割裂感。

不过，使用 Hexo 的直接诱因还是数月前在 V2EX 看到的这篇文章。和 Hexo 默认提供的那几个主题相比， Icarus 干净、明快的风格直戳我心，可以说是站长目前为止见过的最漂亮的博客主题。发现这篇文章时就下定决心切换到 Hexo，到寒假终于找到时间，如愿以偿。

之前和朋友闲聊时统计了一下一只萌新的情况，2017 年更新了 29 篇，2018 年只更了 5 篇。这个数字实在不好看，我也是问心有愧的——去年并没有花什么心思在博客上，有时候想到该写博客却又去做其他事情了。换用 Hexo 后，尽管搭建过程复杂了些，但写作却变简单了，只要新建文件——码字——发布即可。没有了 Wordpress 各种插件、格式等的干扰，争取回归写作本身。

当然，没有丰富的插件必然也会导致一些不便，例如评论。Icarus 提供了多种评论插件，本站目前使用的是 Valine。其优点是干净简洁，缺点则是评论通知非常不便；而 DesertsP 开发的 Valine-Admin 能很好地解决这个问题，然而需要向国内公司做实名认证，故目前暂时没有使用的想法。若需要回复的评论长时间没有得到回复，请直接猛击左侧的『联系我』！

最后，一只萌新使用的图库这几天似乎有些不稳定的迹象，以前发布的文章有些图能正常显示，有些则不能。再观察一段时间，若问题一直持续则会切换图库。（这也是 Hexo 的一个好处，在 Wordpress 上更改文章需要更改本地和已上传的双份，而 Hexo 只需要更改 Markdown 文件，然后重新发布即可）

噢，one more thing。 其实大约一周前一只萌新就已经完成了改版，但为什么今天才发更新说明呢？当然是因为家里没有酒喝啊！站长已经习惯了伴随着一罐啤酒写文章，微醺之下文笔似乎也更加顺畅了。
