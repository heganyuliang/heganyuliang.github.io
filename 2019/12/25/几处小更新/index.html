几处小更新

又是失踪人口回归系列。

闲来一算，一只萌新是在大一寒假时候建的，当时觉得这个名字挺贴切，大一萌新嘛。转眼间还没反应过来，就已经成了大四老狗，甚至研也考完了（虽然基本确定考不上就是了）。因此过了小半年，又有时间写写博客了。

几周之前，突然感觉本站的域名太长了，除了自己以外任何人都难以记住，所以打算买个短点的域名来用，顺便还可以以后当作自己的域名邮箱（假设我叫张三，那么用 i@zhangsan.me 当作简历上的邮箱该有多酷）。但思来想去，始终想不到一个合适的域名，因此尽管花 0.99 刀买了一个 .xyz 域名，但新鲜感过去之后也就迅速抛弃了它。今后本站的域名大概率还是会长期不变，拥有一个真名邮箱的梦想也还是遥遥无期（我的名字常见后缀域名都被韩国某公司注册了，让人气愤）。

不过既然这篇文章标题叫几处小更新，那么一只萌新确实也发生了一些变化。首先是页面左侧的个人信息标签，本站使用的 Icarus 主题更新之后，『联系我』变成了『关注我』。项目作者的 Demo 上指向的是他的 GitHub 页面，而我基本不主动使用 GitHub，并且在配置文件里翻了许久也没找到改回来的方法，因此就把这个按钮指向了我的 Telegram 频道。毕竟之前一直挂着我的邮箱，但也从来没有人单独发邮件，可以认为是没有什么用的。现在换成了 Telegram 频道，大家走过路过可以随手关注一下，我在上面更东西比博客勤多了！（就是随手发一些屁话而已）

此外，可能更有用的更新是页面右上角的一朵小云。前些日子看到了这篇文章，立马动手照葫芦画瓢搭建了一个网盘，现在已经上传了一些电影和电视剧。

FODI 这款程序相当不错，在上面的影音可以直接播放。不过因为后端是 OneDrive 的缘故，在国内能够访问但不是很快，如果要直接在线播放还是确保通过代理比较好。另外我的观影习惯是尽量使用外挂字幕，而 FODI 并不能如本地播放器一样自动加载同一目录下的字幕文件，因此如果像上图那样，一个文件夹里同时有字幕和视频的，那视频一定是生肉，想看熟肉需要把两个文件都下载到本地观看。不过我最近找到一款好用软件（下次更新有内容了），可以很方便地压制 MKV，目前已经把字幕和视频严重不同步的一部分电影（例如红心国王）调整好时间轴压成 MKV 上传了，在线就可以观看有字幕的版本。剩下的大部分电影、以及工作量更大的电视剧，还是等我有心情的时候再压吧。

此外，以后如果在博客上有什么东西分享，也就可以直接上传到这里，不再需要 Google Drive、蓝奏云一类的了。

最后一项可能都注意不到的更新，那就是本站的图片已经全面转向使用腾讯云 COS 自建的图床。之前一直使用的是第三方图床，总有一点担心要是突然倒闭，网站上所有图片都会消失（是的我没有本地备份，我知道这个习惯非常不好），因此还是在前段时间自建了腾讯云北京 COS 存储，并在前两天把所有图片都转移到了上面。尽管没有使用 CDN，但和之前的图床相比，腾讯云的速度在大陆各地表现也都应该足够优秀。当然，这是要花钱的，但就以本站的图片数和访问量来说，每个月几毛一块的支出完全在承受范围之内。用点小钱换来自己对自己东西的掌控感，还是比较安心的。
