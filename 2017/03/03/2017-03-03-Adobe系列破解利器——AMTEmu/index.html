Adobe系列破解利器——AMTEmu

Adobe 系列软件因为功能强大而在很多人生活中是不可或缺的，其价格也十分对得起功能，只是对于个人用户、尤其是学生而言，就显得很不友好了，所以在使用过程中很多人难免会去找破解版。然而，网络上各色人等自制的修改版在安全性方面多少有些让人不放心，并且也不知道是否会打着精简的旗号删改一些有用的东西。因此，最好的办法还是下载原版软件+广受认可的专门破解工具。下面介绍的就是来自毛熊国 Ru-Board 论坛的 painter 大神的工具，AMTEmu。

AMTEmu 通过某种站长也不明白的原理，针对 amtlib.dll 进行破解，从而达到欺骗 Adobe 验证机制的效果。长长的支持列表包括：

使用方法：

先以试用方式安装 Adobe 软件，然后在 AMTEmu 里选择需要破解的软件，单击 Install，找到软件安装目录，双击 amtlib.dll 即可。

站长试图找到 painter 的原帖，结果大概因为该论坛是俄语的缘故，查找许久也一无所获。后来翻到了 painter 的推特，在里面找到了原帖，可是无法打开，翻译一下大概是游客无权查看……总之，原帖打不开的情况下就只好在国内论坛找到了转载的下载地址：

win版：http://ftpportal.tk/PainteR/ru-board/CC/amtemu.v0.9.2.win-painter.zip

mac版：http://ftpportal.tk/PainteR/ru-board/CC/amtemu.v0.8.1.mac-painter.dmg

注意：据转载，原帖为 mac 用户列出了如下一段话：

I had no time to implement the elevation request. That’s why you need to use sudo when you have no sufficient privileges to modify the apps folder. Otherwise you will need to create configs and copy emulator’s files manually.

谷歌机翻一下是：

我没有时间实现高程请求。 这就是为什么当您没有足够的权限修改应用程序文件夹时，您需要使用 sudo。 否则，您将需要手动创建配置和复制仿真器的文件。

在这之下列出了几段命令，应该是解决方式：

Last login: Mon Oct xx xx:xx:xx on console
User:~ user$
sudo /AMTEmu.app/Contents/MacOS/AMTEmu

站长从未使用过 mac 设备，手边也没有可供测试的机器，所以 mac 用户还需自测。

若是觉得国外链接速度太慢，可下载站长上传的网盘版本：

蓝奏云
