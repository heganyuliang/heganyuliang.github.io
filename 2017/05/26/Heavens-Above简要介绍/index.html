Heavens-Above 简要介绍

本篇文章最早发布在我校天文社公众号上，作者也是我，所以没有转载信息……

Heavens-Above 是一个由德国人 Christopher Peat 创建并维护，免费提供多种天文信息的网站。

进入网站后，右上角默认的地址为经纬 0 °，位于几内亚湾的某地，所以我们需要首先在设置里选择自己所在的地区，同时也可以先在右上角设置语言。点击“更改您的观测位置”，在打开的页面有搜索、地图拖动和手动输入信息三种设置方式。地图使用的似乎是谷歌地图中国版的数据源，所以不需要梯子即可正常使用。

当选择好自己所处的地区的时候，就可以查看各式各样的信息了！先来挑个好玩的，我们的天宫二号：

可以看到，这个页面显示了最近十天在北京可见的天宫二号所有过境情况，亮度、时间、方位等信息一应俱全，大家可以依靠这个去观测天宫二号划破天际时的闪光。不过从表上看，最近它经过北京的时间都是凌晨，想要看到它的人恐怕需要很强的毅力了。至少站长是没有的。

相似地，Heavens-Above 还提供了诸如铱星闪光、哈勃望远镜过境等信息，这里就不逐一描述了。除了人造天体外，网站上还有许多自然天体的信息，比如日食：

这个页面列出了最近几年会出现的各种日全食，每个都能点击进入详情页面查看具体情况。虽然目前这个页面显示的日食图看似复杂，但结合详情页面的交互式动画，很容易就能理解图上各种线条的含义。百闻不如一见，站长在这里不多说，感兴趣的朋友还请自行操作，一定会让你有豁然开朗之感。

最后放一个逼格特高的东西，国际空间站三维模型：

国际空间站，常缩写为 ISS，是由美国国家航空航天局、俄罗斯联邦航天局、日本宇宙航空研究开发机构、加拿大太空局、巴西航天局和欧洲空间局六家机构共同运营的近地空间站。这个页面的 ISS 三维模型依照空间站的真实位置不断运动，并且可以随意旋转、放大与缩小，方便浏览者从各个角度观察。站长在上面发现了 NASA，ESA 与 NASDA 的标志，不知道你能不能发现更多呢？

最后再放一张由“发现”号航天飞机拍摄的国际空间站。在许多人对航天事业提出质疑的时候，站长始终坚信，航天是当之无愧的人类智力的最高峰，是人类理性与情感最完美的结合。正如这个网站名所表示的，举头仰望时，人类总能望见令人一处伟大的天堂。
