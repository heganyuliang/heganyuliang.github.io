简明安卓刷机准备流程

要说安卓与 ios、wp 等系统相比最大的特点那就是高度的自定义化，用户通过一系列设置能够随心所欲地对自己的手机进行设置。这篇文章就简单地谈一谈安卓通用的刷机前期准备流程。

注意：由于不同型号手机各自的特殊性，并不存在一种适用于所有手机的方法或工具，因此这里只会涉及各步骤的原理，至于所需要的具体工具，请到各自品牌的论坛查询，如MIUI论坛等，或者是直接前往国外的 xda 论坛——这才是真正大神云集的地方，诸如 TWRP、SuperSu 等都是在这里发布再被转载到国内的。

零、设置开发者选项

之前把这段漏了……因为手机到手后就打开了再也没关过，所以没想起还有这步。

首先打开设置-关于手机，连击7次版本号打开开发者选项，然后退回到上级菜单，进入开发者选项，将OEM解锁和USB调试都打开，否则是无法刷机的。

一、解锁 BootLoader

BootLoader 与电脑的 BIOS 类似，是在系统启动前进行硬件初始化，获取内存大小信息等，调整手机到适配状态等工作的一段程序。这种非常底层的东西自然十分重要，大部分厂商都会为其加锁。而为了后续操作，我们有必要对其进行解锁。解锁的方法多种多样，许多论坛如著名的移动叔叔都提供工具箱一键解锁，不过我更建议按照 xda 上 TWRP 发布者的教程来解锁。

二、刷入第三方 Recovery

Recovery 常简称 rec，是安卓的一个恢复系统，类似于 Windows 的 pe，可以在关机状态下通过同时长按电源键和音量减来进入。手机出厂时就自带官方 rec 了，不过无法刷入任何非官方包，因此需要刷入第三方 rec。第三方 rec 主要有 CWM 和 TWRP，两家曾经成分庭抗礼之势，不过 TWRP 由于支持触摸操作等原因而在最近渐渐超越了 CWM，更为流行，并且自从 3.0.0-1 后原生加入了简体中文支持，国内用户也不再需要去找各种中文版了。

以站长的一加 3 为例，这里给出 xda 上的 TWRP 贴子，可以看到都是非常简单的英文，只有少量的安卓术语，不过也都是刷机过程中的常见词汇，极易掌握。

这里特别给出安卓主要版本的中英文代号，以防下载了错误的工具版本：

7.x: Nougat(N) 牛轧糖

6.x: Marshmallow(M) 棉花糖

5.x: Lollipop(L) 棒棒糖

4.4.x: KitKat(KK) 奇巧巧克力

更古老的安卓版本现在很少见了，因此也就不展开描述，若有需要请参考维基百科。

另外还需注意的是，一些手机可能存在特别情况，如站长的一加 3 在刷入第三方 rec 后会触发 dm-verity，需要有一些特殊处理。在 xda 上通常会有说明，请仔细阅读，并严格按照上面的流程操作。

三、root

是否 root 其实是一个见仁见智的问题。root 后，顾名思义，用户就能掌握系统的最高权限（根权限），能对系统进行任意修改。然而，一旦有害应用诱导用户给自己授予了 root 权限，它就能够对你的手机为所欲为。前不久才发生了有人修改的绿色守护获取权限后伪装成微信诱导用户输入支付密码，造成财产损失的事件。因此，是否 root 以及 root 后是否给应用授予权限，需要慎重考虑。

至于 root 方法，也是种类繁多。Kingroot 之流有后台安装其他软件的流氓行为，而曾经绝对的老大 SuperSu 几个月前被一个传闻来自中国的神秘公司收购，让许多重视隐私的用户深感担忧，毕竟国内互联网公司算得上是劣迹斑斑，而将 root 权限这么重要的东西交予国内公司实在令人难以放心。因此，这里介绍一个新型的 root 方法：Magisk。

其实将 Magisk 当作 root 方法完全是大材小用，它的最初目的是在不修改 /system 文件夹的情况下达到修改的效果，root 只是产生的一个附带功能，因为传统的 root 是修改 /system 中的文件。另外，Magisk 完全开源，其安全性可以放心。刷入 Magisk 的方法是通用的，详情请参考上面给出的 xda 链接，这里只做简单的介绍：

  1. （建议）备份 boot.img（在第三方 rec 中操作）；
  2. 下载最新版 Magisk 的卡刷包，放入手机根目录；
  3. 如果想要使用 SuperSu，以Systemless 模式刷入 SuperSu（只支持安卓6.0以上）；否则跳过这步；
  4. 刷入 Magisk.zip；
  5. 重启，安装最新版 Magisk Manager。

值得一提的是，Magisk 的作者 topjohnwu 是来自台湾的学生，算是 xda 上难得的中国开发者，它的脸谱标志也极富中国韵味。关于 Magisk 的其他应用，之后站长会专门写一篇介绍。

至此，你的手机已经完成了搞机的准备工作，胆大心细地去研究安卓吧！最后给出几个有用的参考：

xda论坛：一切技术的源泉。

酷安：站长对于安卓的一切知识几乎都来自酷安。酷安正在经历改版，社区化的应用市场定位很适合交流技术，不过老用户也产生了劣币驱逐良币的担忧。希望新加入酷安的萌新不要成为劣币。

wuxianlin：吴大，国内著名大神，分享了很多针对一加、oppo、flyme 等的定制工具。
