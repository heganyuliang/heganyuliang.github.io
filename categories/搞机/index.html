再次更改评论系统

今天收到之前使用 Valine 评论时后端服务商 LeanCloud 的邮件，随意翻了一下，发现他们提到了加快国内外分开运营。灵机一动，在他们的国际版网站注册了一个账号，果然没有强制实名认证，并且用来定时唤醒的 Web 主机域名也不需要实名认证了（之前使用国内版时，就因为要实名认证一直没有做这个，导致有评论也很难收到通知虽然也没有就是了）。马上行动，好看更好用的 Valine 评论系统又回来了！☟

小白浅谈Xposed与Magisk

前一篇文章说要介绍 Magisk，想了想干脆把 Xposed 一起介绍了吧。

Xposed 无疑是站长接触安卓以来见过的最强力的东西，曾经为了使用它而坚决不上 7.0（虽然后来还是叛变革命了）。Xposed 框架由 xda 大神 rovo89 开发，通过替换安卓系统 /system/bin 目录下的文件，实现对系统某些功能的接管，进而给予基于 Xposed 框架开发的 app（Xposed 模块）更多权限。用户可以通过安装模块深度定制自己的安卓系统，实现诸如 qq、微信防撤回、为单独应用修改 dpi 或语言、强行阻止毒瘤常驻后台等功能。
